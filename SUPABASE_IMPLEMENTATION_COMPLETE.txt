DATABASE SCHEMA
================================================================================

The "recipes" table has these columns:

  id              UUID (auto-generated, primary key)
  title           TEXT (recipe name)
  ingredients     JSON array (list of ingredients)
  instructions    TEXT (cooking instructions)
  notes           TEXT (tips and memories)
  photo           TEXT (base64-encoded image)
  language        TEXT (original language: en, fr, ja, zh-CN)
  translations    JSON object (translations in other languages)
  created_by      TEXT (Little B, Little Pan, etc.)
  created_at      TIMESTAMP (when created)
  updated_at      TIMESTAMP (when last updated)

Row Level Security (RLS) is enabled with open policies — suitable for 
a private family app. If you later want multi-user auth, those policies 
can be tightened.

Real-time subscriptions are enabled so changes sync instantly.

SYNC STATUS INDICATOR
================================================================================

The app shows sync status in the top-right corner:

  ✓ Synced    (green)  → Everything is up to date
  ⟳ Syncing   (amber)  → Currently syncing with Supabase
  ⊗ Offline   (gray)   → No Supabase connection (working offline)
  ! Error     (red)    → Sync error (check browser console F12)

HOW TO TROUBLESHOOT
================================================================================

1. Sync status says "Offline"
   → Check that .env.local exists with correct URL and key
   → Check your internet connection
   → Check browser console (F12) for error messages

2. Recipes not syncing to Supabase
   → Look at sync status indicator
   → Open browser console (F12 → Console tab)
   → Look for red error messages
   → Check .env.local has correct credentials

3. Real-time updates not working
   → Refresh the page
   → Make sure SQL migration ran successfully (step 2 above)
   → Both apps need to be running for you to see updates

4. Can't find Supabase credentials
   → Go to your project dashboard
   → Click Settings (⚙️ icon)
   → Click "API"
   → URL and Anon Key are right there
